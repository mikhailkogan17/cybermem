# API Reference

CyberMem exposes a simple HTTP API for memory operations directly (without MCP).

## Base URL

`http://localhost:8080` (Default)

## Endpoints

### `GET /metrics`

Returns system metrics in Prometheus format.

### `POST /v1/memory`

Create a new memory.

**Body:**

```json
{
  "content": "User prefers dark mode",
  "metadata": {
    "category": "preferences"
  }
}
```

### `GET /v1/memory/search`

Search memories.
**Query Params:**

- `q`: Search query
- `limit`: Max results (default 10)

### `DELETE /v1/memory/:id`

Delete a specific memory by ID.
