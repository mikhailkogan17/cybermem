# MCP Integration Guide

The Model Context Protocol (MCP) allows AI coding assistants to connect directly to CyberMem.

**Detailed connection instructions for all supported clients (Claude, Cursor, VS Code, etc.) are available directly in your CyberMem Dashboard.**

1. Open your CyberMem Dashboard (default: `http://localhost:3000`).
2. Click on the **MCP Config** button in the top right corner.
3. Select your AI client to view specific configuration snippets.

## Quick Reference

- **Server URL:** `http://localhost:8080/mcp`
- **Transport:** SSE (Server-Sent Events)
