## 3. Technical Audit & Fixes

- **Standardized API Schemas**: Fixed mismatches in `apiKey` vs `token` fields and `sparklines` requirements for hydration.
- **Environment Isolation**: UI tests now run strictly mocked against `npm run dev`, fulfilling the zero-Docker requirement.
- **Identity Compliance**: Every component now explicitly identifies as `antigravity-client`.
- **MCP Config Fixed**: Identified and resolved a regression where `/api/mcp-config` failed on RPi due to unreliable internal fetches. Now uses `localhost:3000` internal port for self-calls.
- **Clean Install Recovery**: Verified system stability from absolute zero state (purged volumes + purged `~/.cybermem`).
